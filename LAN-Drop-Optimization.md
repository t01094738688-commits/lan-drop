# LAN Drop 项目优化与发布执行指南

## 一、项目定位

LAN Drop 是面向个人用户的局域网工具箱，核心用途是：

* 手机和电脑在同一 Wi-Fi 下快速互传文件、图片和文字
* 不需要登录账号或安装手机 App
* 文件默认保存在本地
* 支持随记和钥记功能，增加个人工具箱差异化

**版本定位：** v0.1.0-beta 早期测试版
**适用场景：** 小范围测试、GitHub 展示、朋友间使用
**不适用场景：** 大规模公开推广、存放唯一重要账号密码、公共 Wi-Fi 传输敏感数据

---

## 二、核心优势

1. **手机无需安装 App**：扫码即可访问电脑端
2. **路径简单**：电脑端打开 -> 手机扫码 -> 传输
3. **文件/图片/文字均可传**：不仅是剪贴板
4. **差异化功能**：互传、随记、钥记功能组合
5. **对小白用户友好**：不登录、不复杂操作

---

## 三、当前不足

1. 缺实时剪贴板同步功能（UniClipboard）
2. 跨平台发布不完整（Windows / Mac / Linux / iPhone / Android）
3. Mac/Linux 未充分真机测试
4. 没有自动更新
5. 钥记功能敏感，需要备份和主密码说明
6. 设备管理功能不够完善（缺“踢出设备/信任设备”操作）

---

## 四、UI 改进建议

1. **顶部状态区**

   * 显示软件状态（已启动/未启动）、当前网络、手机访问地址、访问码、二维码
   * 提示文案示例：

     ```text
     已启动，手机和电脑连接同一个 Wi-Fi 后即可互传
     未启动，请检查网络或端口占用
     ```

2. **手机访问卡片**

   * 显示手机访问地址、访问码、二维码
   * 添加按钮：复制地址、复制访问码、刷新访问码

3. **新手使用步骤卡片**

   1. 手机和电脑连接同一个 Wi-Fi
   2. 手机扫码或打开访问地址
   3. 输入访问码
   4. 选择文件、图片或文字发送

4. **发送区域优化**

   * 标题：发送文件到手机/电脑
   * 说明：拖入图片、文件，或点击按钮选择。支持 Ctrl+V 粘贴截图
   * 按钮：选择文件/图片、发送文字
   * 拖拽文件时高亮显示

5. **收到内容区域**

   * 卡片化显示：文件类型图标、文件名、时间、大小、来源信息、操作按钮
   * 操作按钮文案统一：打开、复制、删除
   * 空状态提示：

     ```text
     还没有收到内容
     从手机发送文件、图片或文字后，会显示在这里
     ```

6. **功能入口说明**

   * 互传：手机和电脑互传文件、图片、文字
   * 随记：临时记录文字、链接、想法
   * 钥记：保存临时重要信息，提醒用户备份

7. **视觉风格**

   * 主色绿色、浅灰背景、白色卡片、圆角、轻微阴影
   * 字体层级清晰
   * 删除按钮使用危险色或描边样式
   * 留白均匀

1. 文件传输

   * 大文件上传进度提示

   * 中文文件名支持

2. 文字传输

   * 长文本、表情、链接支持
   * 复制文字功能正常

3. 访问码 / 二维码

   * 校验正确

   * IP 地址变化二维码更新
   * 无可用局域网 IP 提示

4. 删除逻辑

   * 删除内容前二次确认
   * 避免误删

5. 钥记功能

   * 明显安全提醒
