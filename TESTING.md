# LAN Drop 测试清单

此清单用于 `v0.1.4-beta` 阶段的小范围测试。

## 基础测试

- Windows 能正常启动。
- 打开软件后能看到手机访问地址、访问码和二维码。
- 手机和电脑连接同一个 Wi-Fi 后，手机扫码能打开页面。
- 输入正确访问码后能进入页面。
- 输入错误访问码时能提示错误。
- 手机发送图片到电脑，电脑能收到。
- 手机发送文件到电脑，电脑能收到。
- 手机发送文字到电脑，电脑能收到。
- 电脑发送文件到手机，手机能下载或打开。
- 电脑发送文字到手机，手机能复制。
- 电脑读取剪贴板文字/链接并发送到手机。
- Ctrl + V 粘贴截图后能进入发送队列。
- 已连接设备列表能显示手机/浏览器。
- 踢出设备后，该设备需要重新输入访问码。
- 收到内容可以删除。
- 文件名包含中文时能正常显示。

## 随记测试

- 可以新增一条文字随记。
- 可以给随记添加图片。
- 文字和图片能作为同一条随记保存。
- 可以按日期查看随记。
- 可以按分类筛选随记。
- 可以搜索随记。
- 可以编辑、删除、复制、分享随记。

## 钥记测试

- 第一次可以创建主密码。
- 关闭后再次打开，正确主密码可以解锁。
- 错误主密码不能解锁。
- 可以新增、编辑、删除一条记录。
- 可以复制密钥/密码。
- 如果忘记主密码，清空本机库前有明确提醒。

## 兼容测试

- iPhone Safari。
- Android Chrome。
- Windows Chrome / Edge。
- Mac Safari。
- 微信内置浏览器：如果体验异常，应提示使用系统浏览器。

## 异常测试

- 手机和电脑不在同一个 Wi-Fi。
- 电脑端口被占用。
- Windows 防火墙拦截。
- 手机扫码后 IP 地址不可访问。
- 上传大文件。
- 上传过程中断网。
- 文件名重复。
- 文件名包含空格、中文、特殊符号。
