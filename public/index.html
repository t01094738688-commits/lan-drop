发送文件到手机/电脑

拖入或选择文件，也可以复制文件后按 Ctrl + V 直接发送。手机端可选择相册或文件发送到电脑。

使用方法

  1. 手机和电脑连接同一个 Wi-Fi
  2. 手机扫码或打开访问地址
  3. 输入电脑端显示的访问码
  4. 选择文件、图片或文字，内容会出现在互传列表
