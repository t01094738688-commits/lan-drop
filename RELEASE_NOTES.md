# LAN Drop v0.1.38-beta

这是 LAN Drop / 闪传本子的早期测试版。

## 下载

请前往 GitHub Releases：

[https://github.com/t01094738688-commits/lan-drop/releases](https://github.com/t01094738688-commits/lan-drop/releases)

当前计划提供：

- Windows x64：`.exe`
- macOS Apple Silicon：`.dmg` / `.zip`
- macOS Intel：`.dmg` / `.zip`
- Linux：`.AppImage` / `.deb`

## 本版本重点

- 软件内更新改为“下载并安装”：不再弹浏览器保存窗口，更新包会下载到本地 `updates` 目录，下载完成后自动打开安装包并退出旧版。
- 更新过程中显示下载进度和失败原因，避免长时间空白。
- 钥记打开后，本次软件运行期间会一直保留在左侧导航，不用每次从设置的高级功能里重新进入。
- 保留“手机免安装，扫码即用”的核心体验。
- 支持文件、图片、文字互传。
- 电脑复制文件或图片后，可在互传页按 `Ctrl + V` 直接发送。
- 随记可保存重要文字和图片。

## 已知限制

- 这是早期测试版，Mac/Linux 仍需要更多真机测试。
- macOS 版本尚未做 Apple 签名/公证，首次打开可能需要右键点击应用并选择“打开”。
- Linux 包属于实验测试版，不同发行版表现可能不同。
- 软件内更新会自动下载并打开安装包，但还不是完全无感静默覆盖。
- 手机浏览器不能后台自动读取手机剪贴板，手机端剪贴板同步必须由用户主动触发。

## 使用提醒

- 手机和电脑需要连接同一个 Wi-Fi 或同一个热点。
- 建议在可信网络中使用，不建议在公共 Wi-Fi 传输敏感文件。
- 手机端不需要安装 App，使用 Safari、Chrome 等系统浏览器即可。
