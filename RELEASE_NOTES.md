# LAN Drop 0.1.0

## 给别人用哪个文件

- 直接试用：`release/LAN Drop 0.1.0.exe`
- 长期安装：`release/LAN Drop Setup 0.1.0.exe`

## 当前版本能力

- 手机上传文件到电脑。
- 电脑拖拽文件或粘贴截图/文字给手机。
- 图片一键复制到 Windows 剪贴板。
- 钥记：只填名称和密钥/内容，本地加密保存。
- 随记：保存临时文字和图片。
- 最小化到系统托盘。

## 当前限制

- 默认只支持同一个局域网。
- 钥记和随记数据保存在当前浏览器/软件本地，换设备前需要导出或迁移。
- 还没有正式图标和签名，Windows 可能会提示未知发布者。
