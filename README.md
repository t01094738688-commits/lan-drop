# 闪传本子

闪传本子是一个本地运行的手机电脑互通工具。电脑端启动后，手机在同一个 Wi-Fi 下打开访问地址，就能直接传截图、图片、文件和文字到电脑。

## 功能

- 互传：手机和电脑互传截图、图片、文件和文字。
- 钥记：本地加密保存账号、邮箱、密钥、密码、Token 等敏感内容。
- 随记：临时记录文字和图片，适合放网页测试素材、截图和零散想法。
- 访问码：手机访问需要输入本机生成的访问码，避免同网段陌生设备直接查看内容。
- 桌面版：支持托盘、关闭确认、安装前清理旧进程。

## 开发运行

```powershell
npm install
npm run desktop
```

只运行网页服务：

```powershell
npm start
```

默认端口是 `47321`。

## 打包

便携版：

```powershell
npm run pack:win
```

安装版：

```powershell
npm run dist:win
```

生成结果在 `release` 文件夹。

## 用户使用提醒

- 手机和电脑需要在同一个 Wi-Fi 或同一个热点网络下。
- 如果手机打不开访问地址，优先检查 Windows 防火墙和网络是否为“专用网络”。
- 钥记的主密码不会上传，也无法找回；忘记密码只能清空本地库。
- 默认数据保存在用户文档目录下的 `闪传本子` 文件夹。
